# S1 - Layer-2 Switch (Physical Device: Cisco Switch 2960)

This file documents `S1`, the Layer‑2 switch used in the **Secure Enterprise Network** project. `S1` is the virtual representation of a **Cisco 2960‑style switch** in Cisco Packet Tracer and connects two departments: **Computer Sciences** (VLAN 10) and **Data Science** (VLAN 20).

## Physical device details

- **Real‑world model**: Cisco Catalyst 2960  
- **Virtual model in Packet Tracer**: "Switch" / 2960‑style  
- **Location**: Access layer, Department 1 area (Computer Sciences + Data Science)

## Configuration

```bash
hostname S1
!
vlan 10
 name CS-Dept
!
vlan 20
 name DS-Dept
!
interface range fa0/1 - 10
 switchport mode access
 switchport access vlan 10
!
interface range fa0/11 - 20
 switchport mode access
 switchport access vlan 20
!
interface fa0/24
 switchport mode trunk
 switchport trunk allowed vlan 10,20
 spanning-tree portfast trunk
!
end
```

- Ports `fa0/1–10` → **Computer Sciences** (VLAN 10).  
- Ports `fa0/11–20` → **Data Science** (VLAN 20).  
- `fa0/24` → **trunk** to `CoreSwitch`.
