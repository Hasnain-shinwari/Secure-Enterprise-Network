# S2 - Layer-2 Switch (Physical Device: Cisco Switch 2960)

`S2` is the second Layer‑2 switch in the **Secure Enterprise Network**. It is the virtual representation of a **Cisco 2960‑style switch** and connects **Cyber Security** (VLAN 30) and **Software Eng.** (VLAN 40).

## Physical device details

- **Real‑world model**: Cisco Catalyst 2960  
- **Virtual model in Packet Tracer**: "Switch" / 2960‑style  
- **Location**: Access layer, Department 2 area (Cyber Security + Software Eng.)

## Configuration

```bash
hostname S2
!
vlan 30
 name CSec-Dept
!
vlan 40
 name SWE-Dept
!
interface range fa0/1 - 10
 switchport mode access
 switchport access vlan 30
!
interface range fa0/11 - 20
 switchport mode access
 switchport access vlan 40
!
interface fa0/24
 switchport mode trunk
 switchport trunk allowed vlan 30,40
 spanning-tree portfast trunk
!
end
```

- Ports `fa0/1–10` → **Cyber Security** (VLAN 30).  
- Ports `fa0/11–20` → **Software Eng.** (VLAN 40).  
- `fa0/24` → **trunk** to `CoreSwitch`.
