# S4 - Layer-2 Switch (Physical Device: Cisco Switch 2960)

`S4` is the fourth Layer‑2 switch in the **Secure Enterprise Network**. It is the virtual representation of a **Cisco 2960‑style switch** and connects **Admin** (VLAN 70) and **HR** (VLAN 80).

## Physical device details

- **Real‑world model**: Cisco Catalyst 2960  
- **Virtual model in Packet Tracer**: "Switch" / 2960‑style  
- **Location**: Access layer, Department 4 area (Admin + HR)

## Configuration

```bash
hostname S4
!
vlan 70
 name Admin-Dept
!
vlan 80
 name HR-Dept
!
interface range fa0/1 - 10
 switchport mode access
 switchport access vlan 70
!
interface range fa0/11 - 20
 switchport mode access
 switchport access vlan 80
!
interface fa0/24
 switchport mode trunk
 switchport trunk allowed vlan 70,80
 spanning-tree portfast trunk
!
end
```

- Ports `fa0/1–10` → **Admin** (VLAN 70).  
- Ports `fa0/11–20` → **HR** (VLAN 80).  
- `fa0/24` → **trunk** to `CoreSwitch`.
