# S3 - Layer-2 Switch (Physical Device: Cisco Switch 2960)

`S3` is the third Layer‑2 switch in the **Secure Enterprise Network**. It is the virtual representation of a **Cisco 2960‑style switch** and connects **Artificial Int.** (VLAN 50) and **IoT** (VLAN 60).

## Physical device details

- **Real‑world model**: Cisco Catalyst 2960  
- **Virtual model in Packet Tracer**: "Switch" / 2960‑style  
- **Location**: Access layer, Department 3 area (Artificial Int. + IoT)

## Configuration

```bash
hostname S3
!
vlan 50
 name AI-Dept
!
vlan 60
 name IoT-Dept
!
interface range fa0/1 - 10
 switchport mode access
 switchport access vlan 50
!
interface range fa0/11 - 20
 switchport mode access
 switchport access vlan 60
!
interface fa0/24
 switchport mode trunk
 switchport trunk allowed vlan 50,60
 spanning-tree portfast trunk
!
end
```

- Ports `fa0/1–10` → **Artificial Int.** (VLAN 50).  
- Ports `fa0/11–20` → **IoT** (VLAN 60).  
- `fa0/24` → **trunk** to `CoreSwitch`.
